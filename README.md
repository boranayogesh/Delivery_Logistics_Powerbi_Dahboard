#  Delivery & Logistics Performance Analysis Dashboard

##  Project Overview

The **Delivery & Logistics Performance Analysis Dashboard** is an end-to-end Business Intelligence solution developed using **Power BI** to analyze shipment operations, delivery efficiency, route performance, and logistics KPIs. This dashboard enables logistics managers and business stakeholders to monitor operational performance, identify bottlenecks, and make data-driven decisions to improve delivery efficiency and customer satisfaction.

The project transforms raw logistics and shipment data into meaningful insights through interactive visualizations, KPI tracking, trend analysis, and operational performance monitoring.

---

##  Business Objective

Logistics organizations handle thousands of shipments daily across different transportation routes and delivery channels. Monitoring delivery performance, route efficiency, and operational compliance becomes challenging when data is spread across multiple systems.

This dashboard was developed to:

* Monitor delivery performance in real-time.
* Track on-time and delayed shipments.
* Analyze route efficiency and distance coverage.
* Compare estimated delivery metrics with actual outcomes.
* Identify operational bottlenecks affecting customer service.
* Improve decision-making through data visualization.

---

##  Key Performance Indicators (KPIs)

The dashboard provides a comprehensive overview of logistics operations through the following KPIs:

* Total Shipments Processed
* Average Delivery Time (Minutes)
* Total Distance Covered
* On-Time Delivery Percentage (%)
* Delayed Shipment Percentage (%)
* Route-wise Shipment Distribution
* Cutoff Compliance Rate
* Delivery Trend Analysis

---

##  Dashboard Features

### Executive Summary

Provides a high-level overview of logistics performance including shipment volume, delivery efficiency, distance covered, and service-level metrics.

### Cutoff Compliance Analysis

Tracks shipment compliance against operational cutoff times and identifies areas impacting delivery performance.

### Route Performance Analysis

Analyzes shipment volume and delivery performance across different route types such as:

* Carting
* FTL (Full Truck Load)

### Delivery Trend Monitoring

Visualizes shipment trends over time to identify peak operational periods and delivery patterns.

### OSRM vs Actual Performance Analysis

Compares:

* Estimated Delivery Time vs Actual Delivery Time
* Estimated Distance vs Actual Distance

This helps evaluate route planning accuracy and identify operational deviations.

### Scan Duration Analysis

Measures shipment processing duration from start scan to end scan to identify process inefficiencies.
